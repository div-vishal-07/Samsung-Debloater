# HyperOS-Miui_Debloater_Plugin-Module
To Use this Module .
First install this module in AX Manager (for non-root)
Or You can install this module in KernelSU/Magisk (for root)
Tap on WebUI icon.
Select Apps ( Ticked ✅ will be Debloated/Uninstalled)
Deselect Apps (Unticked ⬜ will be Re-installed)
Now tap Save and Sync
Tap on Action/Run button 
Done !

If you found you've mistakenly debloated/ Uninstalled an useful app 
Just Deselect ⬜ it It will be Reinstalled 

Miui/HyperOS Debloater module for Google and Xiaomi apps. Uses -k flag so it can preserve app data.
